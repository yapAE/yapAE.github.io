Renderer

Renderer는 내용들을 그릴(render) 때 사용합니다.

개요

인자 설명
name 입력 파일의 확장자 (소문자, ‘.’ 사용 불가)
output 출력 파일의 확장자 (소문자, ‘.’ 사용 불가)
sync 동기(Sync) 모드

두 개의 인자가 render 함수로 전달됩니다:

인자 설명
data 두 개의 속성을 포함합니다: 파일 경로인 path와 파일 컨텐츠인 text. path는 없어도 됩니다.
option 옵션
