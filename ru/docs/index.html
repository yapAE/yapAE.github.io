Что есть Hexo?

Hexo-это быстрый, простой и мощный фреймворк блогов. Вы пишите посты в Markdown (или других языках) и Hexo генерирует статические файлы с красивой темой в считанные секунды.

Для пользователей Mac

Вы можете столкнуться с проблемами при компиляции. Пожалуйста, сначала установите Xcode из магазина App Store. После установки, откройте xcode и перейти в Настройки -> Загрузки -> Командная строка -> Установить (Preferences -> Download -> Command Line Tools -> Install), чтобы установить утилиту командной строки.

Установка Git

  • Windows: Скачать и установить git.
  • Mac: Установите с помощью Homebrew, MacPorts или установочного файла.
  • Linux (Ubuntu, Debian): sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS): sudo yum install git-core
