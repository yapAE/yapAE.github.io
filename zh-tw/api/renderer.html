渲染引擎（Renderer）

渲染引擎用於渲染內容。

概要

參數 描述
name 輸入的副檔名（小寫，不含開頭的 .）
output 輸出的副檔名（小寫，不含開頭的 .）
sync 同步模式

渲染函數中會傳入兩個參數：

參數 描述
data 包含兩個屬性：檔案路徑 path 和檔案內容 text。path 不一定存在。
option 選項
